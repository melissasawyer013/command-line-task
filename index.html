Facts About Command Line

  • I can type commands in my terminal to make my computer do neat things. The following basic commands are helpful for navigation:
    • $ ls : lists all the contents of the directory I'm in
    • $ pwd : outputs the name of the directory I'm currently in
    • $ cd : changes the directory I'm currently in to whatever file path I enter after the $ cd

Facts About Git

  • Git is an industry-standard version control tool used to save all relevant versions of work, making it easy for programmers to collaborate.
  • Git projects have 3 parts:
    • Working Directory: where you make changes to files
    • Staging Area: where files are "put" when they're ready to be committed
    • Repository: where changes are saved
